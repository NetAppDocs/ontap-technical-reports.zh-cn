= ONTAP存储效率技术报告
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
ONTAP提供全面的存储效率功能产品组合。实时数据压缩、重复数据删除和数据缩减相结合、可降低存储成本并最大限度地提高可存储的数据。此外、您还可以利用节省空间的NetApp Snapshot副本、精简配置、复制和克隆技术、将节省的空间成倍增加。
